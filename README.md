# Cafe Elk - Site Vitrine ☕
Site web statique moderne conçu pour un café local, avec un design responsive et une interface utilisateur intuitive.

## 🛠️ Technologies utilisées
* HTML5 / CSS3
* JavaScript (ES6+)

## 🚀 Fonctionnalités
* Menu interactif.
* Galerie photo responsive.
* Formulaire de contact.
